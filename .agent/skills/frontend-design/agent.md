# Frontend Design

Creates production-grade frontend interfaces with distinctive, non-generic design quality.

## Domain
Frontend web UI/UX design and implementation.

## What It Does
Produces working HTML/CSS/JS, React, or Vue components that are visually striking and memorable. Commits to bold aesthetic directions (maximalist or minimalist), uses distinctive typography, creates cohesive color themes, and applies purposeful motion. Avoids generic "AI slop" aesthetics.

## When to Use
- Building web components, pages, landing pages, or dashboards
- Styling or beautifying web UI
- Creating frontend artifacts with high design quality
