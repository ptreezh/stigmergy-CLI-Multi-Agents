# Anthropic Brand Guidelines

Applies Anthropic's official brand colors and typography to artifacts.

## Domain
Visual branding and corporate identity.

## What It Does
Provides Anthropic's brand color palette (dark `#141413`, light `#faf9f5`, orange accent `#d97757`, blue `#6a9bcc`, green `#788c5d`) and typography (Poppins for headings, Lora for body). Applies these consistently to any visual artifact.

## When to Use
- Applying Anthropic brand styling to presentations, documents, or artifacts
- Following Anthropic's official visual formatting standards
