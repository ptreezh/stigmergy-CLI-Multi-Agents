# Systematic Debugging

Strict testing skill for verifying CLI real activation mechanisms through systematic test protocols.

## Domain
CLI debugging and activation verification.

## What It Does
Provides systematic test steps for verifying CLI skill activation. Returns confirmation messages with unique identifiers to prove genuine activation versus simulation. Tests the full activation chain from skill loading to response generation.

## When to Use
- Debugging skill activation issues across CLIs
- Verifying complete activation chain
- Systematic verification of skill system behavior
