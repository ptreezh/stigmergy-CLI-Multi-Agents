# 商业生态图谱构建技能

## 何时使用
当需要基于关键物种和业务关系构建可视化商业生态图谱时使用此技能。

## 核心功能
1. **关系映射**: 基于业务关系映射物种间的连接
2. **流分析**: 标识业务流、资金流和信息流的方向和强度
3. **可视化构建**: 创建直观的生态图谱可视化
4. **动态更新**: 提供动态更新的生态关系可视化
5. **关键节点标注**: 标识生态中的关键物种和重要连接

## 执行步骤
1. 收集关键物种及其业务关系数据
2. 构建物种间关系网络
3. 识别业务流、资金流和信息流
4. 创建可视化图谱
5. 标注关键节点和重要关系
6. 生成商业生态图谱报告

## 输出结果
- 可视化商业生态图谱
- 物种关系矩阵
- 流量分析报告
- 关键节点识别
- 生态结构分析
