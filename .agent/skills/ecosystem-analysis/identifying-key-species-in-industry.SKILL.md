# 关键物种识别技能

## 何时使用
当需要识别特定行业中的核心参与者、关键企业或重要角色时使用此技能。

## 核心功能
1. **关键物种识别**: 识别行业中的核心企业、组织或角色
2. **重要性评估**: 评估物种在生态系统中的重要性和影响力
3. **生态位分析**: 分析物种在商业生态中的定位和作用
4. **影响力范围确定**: 确定物种的影响力范围和辐射能力
5. **竞争优势识别**: 识别关键物种的核心竞争优势

## 执行步骤
1. 基于商业生态分析结果，识别潜在的关键物种
2. 评估各物种的市场份额、收入规模等量化指标
3. 分析物种间的依赖关系和协同效应
4. 确定关键物种及其在生态系统中的地位
5. 生成关键物种识别报告

## 输出结果
- 关键物种清单及排名
- 物种生态位分析报告
- 物种影响力评估矩阵
- 物种竞争优势分析
- 物种间依赖关系图
