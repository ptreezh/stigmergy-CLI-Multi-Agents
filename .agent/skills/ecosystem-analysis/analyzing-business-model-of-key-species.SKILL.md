# 商业模式分析技能

## 何时使用
当需要深入分析关键物种的商业模式，理解其外部业务模式和上下游关系时使用此技能。

## 核心功能
1. **外部业务模式分析**: 分析物种的渠道模式、客户触达方式和服务提供方式
2. **上下游关系分析**: 分析物种的供应链、分销链和服务链关系
3. **收入模式分析**: 分析物种的收入来源和盈利模式
4. **价值主张分析**: 分析物种为客户提供的核心价值
5. **合作伙伴分析**: 识别物种的重要合作伙伴和协作关系

## 执行步骤
1. 收集关键物种的业务数据和运营信息
2. 分析其外部业务模式（渠道、客户触达、服务方式）
3. 梳理其上下游关系（供应商、合作伙伴、分销商）
4. 分析其收入模式和价值主张
5. 评估业务模式的优势和潜在风险
6. 生成商业模式分析报告

## 输出结果
- 商业模式画布分析
- 价值网络关系图
- 收入模式分析报告
- 竞争优势评估
- 潜在风险识别
