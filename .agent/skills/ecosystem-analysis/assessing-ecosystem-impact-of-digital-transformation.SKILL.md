# 生态变化影响分析技能

## 何时使用
当需要评估数字化转型对整个商业生态系统的系统性影响和未来演化趋势时使用此技能。

## 核心功能
1. **系统性影响评估**: 评估数字化转型对整个生态系统的系统性影响
2. **物种关系变化预测**: 预测物种间竞争与合作关系的变化
3. **生态稳定性分析**: 分析生态系统在数字化转型后的稳定性
4. **演化趋势预测**: 预测生态系统的未来演化趋势
5. **新生态位和物种预测**: 预测可能出现的新生态位和新物种
6. **竞争优势重新分布**: 分析数字化转型后竞争优势的重新分布
7. **生态韧性评估**: 评估生态系统的适应性和韧性

## 执行步骤
1. 评估数字化转型对各关键物种的直接影响
2. 分析物种间关系的潜在变化
3. 预测新生态位和新物种的出现
4. 评估整个生态系统结构的稳定性变化
5. 预测生态系统未来的演化路径
6. 分析竞争优势的重新分布
7. 评估生态系统的韧性与适应性
8. 生成生态变化影响分析报告

## 输出结果
- 生态系统影响综合评估
- 物种关系变化预测报告
- 生态稳定性量化分析
- 系统演化趋势预测
- 新生态位识别与机会分析
- 风险评估与应对建议
