# Skill from GitHub

Creates skills by deeply learning from high-quality GitHub projects.

## Domain
Skill authoring through project study.

## What It Does
Searches GitHub for quality projects solving the target problem, presents top candidates for user selection, dives deep into the selected project (README, source code, examples), summarizes the learned methodology, then hands off to skill-creator for encoding. Focuses on learning knowledge, not just wrapping tools.

## When to Use
- Building a skill based on existing proven open-source projects
- Encoding methodologies from successful GitHub repositories
- Discovering battle-tested approaches before implementing
