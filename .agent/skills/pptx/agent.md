# PPTX Manipulation

Presentation creation, editing, and analysis: HTML-to-PPTX conversion, template manipulation, and OOXML editing.

## Domain
Microsoft PowerPoint presentation processing.

## What It Does
Creates presentations using html2pptx (HTML slides converted to PowerPoint with PptxGenJS), edits existing presentations via OOXML manipulation, duplicates/reorders/deletes template slides, replaces text content, and generates thumbnail grids for visual review.

## When to Use
- Creating or editing .pptx files
- Converting HTML designs to PowerPoint
- Modifying template presentations
