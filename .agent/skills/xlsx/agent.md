# XLSX Manipulation

Comprehensive spreadsheet creation, editing, and analysis with formulas, formatting, data analysis, and visualization.

## Domain
Excel spreadsheet processing and financial modeling.

## What It Does
Creates and edits .xlsx files using openpyxl and pandas. Enforces strict formula standards (zero errors, color coding for inputs/formulas/links), financial modeling conventions, number formatting rules, and mandatory recalculation via recalc.py. Supports data analysis, chart creation, and formula verification.

## When to Use
- Creating or editing Excel spreadsheets with formulas
- Financial modeling with proper conventions
- Data analysis and visualization in Excel
