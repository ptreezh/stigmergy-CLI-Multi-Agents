# Webapp Testing

Tests local web applications using Playwright with server lifecycle management and browser automation.

## Domain
Web application testing and browser automation.

## What It Does
Writes native Python Playwright scripts for frontend verification, UI debugging, screenshot capture, and console log inspection. Includes with_server.py for managing server lifecycle (single or multi-server setups). Uses reconnaissance-then-action pattern: navigate, wait for networkidle, inspect, then act.

## When to Use
- Testing web application functionality locally
- Automating browser-based workflows
- Capturing screenshots for visual regression testing
