# Mathematical Statistics

Comprehensive statistical analysis for social science research: descriptive, inferential, regression, ANOVA, factor analysis, and visualization.

## Domain
Quantitative data analysis for social sciences.

## What It Does
Provides a full statistical pipeline: data preprocessing, descriptive statistics, inferential tests (t-tests, ANOVA, chi-square), regression analysis (linear, logistic, ridge, LASSO), multivariate analysis (factor analysis, clustering, PCA), and statistical visualization. Outputs standardized JSON with three-layer structure.

## When to Use
- Statistical analysis of survey or experimental data
- Regression modeling, factor analysis, or hypothesis testing
- Data visualization for statistical results
