# PDF Manipulation

Comprehensive PDF toolkit: text/table extraction, creation, merging, splitting, forms, and OCR.

## Domain
PDF document processing and generation.

## What It Does
Handles all major PDF operations using Python (pypdf, pdfplumber, reportlab) and command-line tools (pdftotext, qpdf, pdftk). Supports OCR for scanned PDFs, watermark application, password protection, image extraction, and form filling.

## When to Use
- Extracting text or tables from PDFs
- Creating or merging PDF documents
- OCR processing of scanned documents
- Filling PDF forms
