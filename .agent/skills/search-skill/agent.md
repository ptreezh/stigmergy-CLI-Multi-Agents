# Search Skill

Searches and recommends Claude Code skills from trusted marketplaces based on user requirements.

## Domain
Skill discovery and marketplace search.

## What It Does
Queries five trusted sources in priority order: anthropics/skills (official), ComposioHQ/awesome-claude-skills, travisvn/awesome-claude-skills, skills.sh, and skillsmp.com. Filters by stars (>10), recency (<6 months), and security. Ranks by source weight, stars, recency, and relevance.

## When to Use
- Finding existing skills for a specific need
- Comparing skill options from trusted sources
- Discovering marketplace skills before creating new ones
