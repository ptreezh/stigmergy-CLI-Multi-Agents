# Business Workflow

Implements multi-step workflow patterns with state management for general operations.

## Domain
Workflow orchestration and process automation.

## What It Does
Executes multi-step processes with state persistence, branching logic, human-in-the-loop approvals, and rollback capabilities. Implements a finite state machine pattern for long-running workflows with timeout and retry mechanisms.

## When to Use
- Orchestrating multi-step business processes
- Managing workflows with conditional branching
- Needing rollback support for failed steps
