# DOCX Manipulation

Comprehensive Word document creation, editing, and analysis with tracked changes, comments, and formatting support.

## Domain
Microsoft Word document processing.

## What It Does
Handles creating new documents (docx-js), editing existing ones (OOXML manipulation), tracked changes and redlining workflows, comment management, and text extraction. Supports PDF conversion and image export for visual analysis.

## When to Use
- Creating, editing, or analyzing .docx files
- Working with tracked changes or comments
- Converting documents between formats
