# ResumeSession

Cross-CLI session recovery and history management for all AI CLI tools with project-aware filtering.

## Domain
Session management and conversation recovery.

## What to Use For
Recovering last conversations, browsing session history, filtering by CLI tool or project, and continuing previous discussions. Reads from CLI-specific session directories and provides cross-platform compatibility (Windows, macOS, Linux).

## When to Use
- "恢复上次对话" or "查看历史会话"
- Resuming a previous project session
- Finding sessions from a specific CLI tool
