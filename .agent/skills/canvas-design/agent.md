# Canvas Design

Creates museum-quality visual art in PNG and PDF using design philosophy movements.

## Domain
Visual design and static artwork creation.

## What It Does
First generates a written design philosophy (an aesthetic movement), then expresses it visually on canvas. Output is PDF or PNG with carefully chosen fonts, colors, and compositions. Each piece is treated as a serious artwork with painstaking craftsmanship.

## When to Use
- Creating posters, static visual art, or design pieces
- User requests original visual compositions
- Museum or gallery quality output needed
