# Strict Test Skill

Strict testing skill for verifying CLI real activation mechanisms with unique identifier confirmation.

## Domain
CLI skill activation verification.

## What It Does
Returns a unique activation identifier. If a CLI successfully loads this skill, it explicitly mentions "strict-test-skill已成功激活" along with the unique ID in its response, proving genuine activation rather than simulation.

## When to Use
- Verifying CLI skill loading across different AI tools
- Confirming the skill system correctly activates skills
- Testing skill integration without ambiguity
