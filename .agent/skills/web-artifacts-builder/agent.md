# Web Artifacts Builder

Builds elaborate multi-component HTML artifacts using React, TypeScript, Tailwind CSS, and shadcn/ui.

## Domain
Complex frontend artifact development.

## What It Does
Initializes a React + TypeScript + Vite project, develops the artifact with shadcn/ui components and Tailwind styling, bundles everything into a single self-contained HTML file using Parcel, and presents it as a Claude artifact. Designed for stateful, multi-component web artifacts.

## When to Use
- Building complex React artifacts with state management or routing
- Creating multi-component frontend artifacts beyond single HTML
- Using shadcn/ui components in Claude artifacts
