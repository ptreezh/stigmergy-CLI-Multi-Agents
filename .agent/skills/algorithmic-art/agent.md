# Algorithmic Art

Creates generative art using p5.js with seeded randomness and interactive parameter exploration.

## Domain
Generative and algorithmic visual art creation.

## What It Does
Combines a written algorithmic philosophy with p5.js implementation to produce unique, reproducible generative artworks. Supports seeded randomness, interactive parameter sliders, and gallery-quality output. Creates original art, not copies of existing artists.

## When to Use
- Users request generative art, algorithmic visuals, flow fields, or particle systems
- Creating interactive HTML art artifacts
- Exploring parametric visual variations
