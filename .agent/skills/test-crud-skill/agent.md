# Test CRUD Skill

Testing variant of the CRUD pattern implementation for data management.

## Domain
CRUD testing and data management testing.

## What It Does
Implements the same CRUD operations (create, read, update, delete) as simple-crud but focused on testing CRUD system behaviors. Validates operation correctness, error handling, and input validation across all CRUD endpoints.

## When to Use
- Testing CRUD system implementations
- Validating data management workflows
- Integration testing for CRUD-based services
