# Network Computation

Social network analysis toolkit: network construction, centrality metrics, community detection, and visualization.

## Domain
Social network analysis and graph computation.

## What It Does
Builds networks from edge lists or matrices, calculates centrality measures (degree, betweenness, closeness, eigenvector, PageRank), detects communities (Louvain, modularity, greedy), identifies structural holes and brokerage roles, and generates static and interactive visualizations. Optimized for Chinese social context (guanxi, mianzi).

## When to Use
- Analyzing social or relational network data
- Identifying key actors through centrality analysis
- Community detection in organizational networks
