# Field Expert (Bourdieu - Expert Level)

Advanced Bourdieu field theory analysis with progressive disclosure, 6-step workflow, and dynamic prompt template loading.

## Domain
Advanced sociological field theory (expert level).

## What It Does
Extends field-analysis with a 6-step sequential workflow: data preparation, boundary analysis, capital analysis, habitus analysis, dynamics analysis, and report generation. Integrates grounded theory, social network, and ESOC framework data. Uses dynamic prompt templates for progressive disclosure.

## When to Use
- Advanced field analysis requiring multiple integrated data sources
- Multi-dimensional analysis combining qualitative and quantitative data
- Generating comprehensive field theory reports with quantitative scripts
