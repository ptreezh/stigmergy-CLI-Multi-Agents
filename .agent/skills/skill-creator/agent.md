# Skill Creator

Guide for creating effective skills following the OpenSkills specification.

## Domain
Skill development and packaging.

## What It Does
Covers the full skill creation lifecycle: understanding skill intent with examples, planning reusable contents (scripts/references/assets), initializing with init_skill.py, editing SKILL.md and resources, and packaging with package_skill.py. Teaches progressive disclosure design principles.

## When to Use
- Creating a new skill from scratch
- Updating an existing skill
- Packaging and validating skill structure
