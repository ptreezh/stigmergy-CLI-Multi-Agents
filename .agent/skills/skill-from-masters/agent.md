# Skill from Masters

Creates high-quality skills by discovering and incorporating expert methodologies before skill generation.

## Domain
Expert methodology-driven skill authoring.

## What It Does
A 10-step process: narrow the skill intent through 5-layer framework, identify skill type (generation, decision, diagnosis, etc.), map to expert domains, research methodologies until crystal clear, find golden examples, select frameworks collaboratively, extract actionable principles, cross-validate, design test scenarios, then invoke skill-creator. For non-technical skills, mandates Opus model.

## When to Use
- Creating skills involving decision-making, communication, or judgment
- Encoding expert methodologies before skill creation
- Building high-quality non-technical skills requiring deep domain knowledge
