# User Management

CRUD-based user management skill implementing standard create, read, update, delete operations.

## Domain
User account and identity management.

## What It Does
Implements full CRUD lifecycle for user entities: account creation, retrieval with filters, updates to profile/status/permissions, deletion, and listing with optional filtering. Follows RESTful API conventions with proper validation and error handling.

## When to Use
- User account management workflows
- Implementing user-related CRUD operations
- Managing user profiles and permissions
