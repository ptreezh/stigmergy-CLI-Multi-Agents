# Theme Factory

Applies curated professional themes (colors + fonts) to artifacts: slides, docs, reports, and HTML pages.

## Domain
Visual theme and styling application.

## What It Does
Offers 10 pre-built themes (Ocean Depths, Sunset Boulevard, Forest Canopy, Modern Minimalist, Golden Hour, Arctic Frost, Desert Rose, Tech Innovation, Botanical Garden, Midnight Galaxy) each with complete color palettes and font pairings. Can also generate custom themes on demand. Applies themes consistently across all slides or artifacts.

## When to Use
- Applying consistent styling to presentations or documents
- Choosing professional color/font themes
- Generating custom themes matching specific aesthetics
