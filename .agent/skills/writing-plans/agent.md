# Writing Plans

Manus-style file-based planning skill for managing complex tasks and research projects.

## Domain
Complex task planning and progress tracking.

## What It Does
Identical to planning-with-files - creates task_plan.md, findings.md, and progress.md for persistent session memory. Uses filesystem as disk vs context window as RAM. Enforces planning-first, error logging, and 3-strike error protocol.

## When to Use
- Multi-step tasks requiring 5+ tool calls
- Research projects spanning multiple sessions
- Any task where persistent notes improve outcomes
