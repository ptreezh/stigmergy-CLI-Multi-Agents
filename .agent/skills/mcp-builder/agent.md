# MCP Builder

Guide for creating high-quality Model Context Protocol (MCP) servers that enable LLMs to interact with external services.

## Domain
MCP server development and integration.

## What It Does
Covers the full MCP server development lifecycle: deep research and API planning, TypeScript (recommended) or Python implementation, tool design with Zod/Pydantic schemas, error handling patterns, comprehensive testing with MCP Inspector, and evaluation question creation. References official MCP specification and SDK documentation.

## When to Use
- Building MCP servers to integrate external APIs
- Creating tools that extend LLM capabilities
- Following MCP best practices for server design
