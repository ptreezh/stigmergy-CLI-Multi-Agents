# Grounded Theory Expert

Complete grounded theory analysis framework: open coding, axial coding, selective coding, memo writing, and saturation testing.

## Domain
Qualitative research methodology and theory construction.

## What It Does
Guides the full grounded theory process: open coding for concept identification, axial coding for category building and paradigm models, selective coding for core category and theory construction, saturation testing for completeness, and memo writing for theoretical reflection. Optimized for Chinese research contexts.

## When to Use
- Building theory from qualitative data
- Processing interview transcripts or field notes
- Conducting systematic qualitative research with coding
