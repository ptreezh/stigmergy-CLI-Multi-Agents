# 研究分歧解决技能 (Research Conflict Resolution)

## 概述

研究分歧解决技能专门处理学术研究过程中出现的各种观点分歧、方法论争议和解释冲突，确保研究团队能够建设性地解决分歧并推进研究。

## 使用时机

当用户请求以下分析时使用此技能：
- 研究中的理论分歧解决
- 方法论争议的处理
- 数据解释冲突的调解
- 基于不同价值观的观点分歧
- 利益相关者的立场差异
- 跨学科合作中的概念分歧
- 学术讨论中的观点对立
- 研究设计中的方法选择争议

## 快速开始

当用户请求分歧解决时：
1. **识别**分歧的类型和根源
2. **分析**各方立场和论据
3. **评估**分歧的严重程度
4. **制定**解决策略
5. **促进**共识建立

## 核心流程

### 第一阶段：分歧识别
   - 识别分歧的核心问题
   - 确定分歧的类型（理论、方法论、解释、价值观、利益）
   - 分析分歧的背景和原因
   - 评估分歧对研究的影响

### 第二阶段：立场分析
   - 明确各方的立场和观点
   - 理解各方的论据和依据
   - 识别共同点和分歧点
   - 分析各方的动机和关切

### 第三阶段：证据整合
   - 收集支持各方观点的证据
   - 评估证据的质量和可靠性
   - 寻找共同认可的证据
   - 识别证据间的联系和矛盾

### 第四阶段：策略制定
   - 选择合适的解决策略
   - 设计建设性对话流程
   - 确定共识建立的目标
   - 制定实施计划

### 第五阶段：共识促进
   - 促进各方对话和交流
   - 引导建设性讨论
   - 帮助各方寻找共同点
   - 协助达成可接受的共识
