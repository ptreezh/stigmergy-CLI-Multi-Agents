# Slack GIF Creator

Creates animated GIFs optimized for Slack using PIL with GIFBuilder utilities, easing functions, and animation concepts.

## Domain
Animated GIF creation for Slack.

## What It Does
Provides GIFBuilder for frame assembly, validators for Slack compliance (128x128 emoji / 480x480 message), easing functions (bounce, elastic, back), and animation concepts (shake, pulse, bounce, spin, fade, slide, zoom, particle burst). Enforces Slack's color (48-128) and FPS (10-30) constraints.

## When to Use
- Creating animated emoji GIFs for Slack
- Making message-level animated GIFs
- Optimizing GIF file size for Slack upload
