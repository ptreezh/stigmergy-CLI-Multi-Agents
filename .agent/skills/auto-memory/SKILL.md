# Auto Memory Skill

自动记忆技能 - 从各个 CLI 的独立运行中汲取经验教训

## 名称
auto-memory

## 描述
在各个 CLI 独立运行时，自动提取会话中的经验教训，更新到共享记忆（STIGMERGY.md），实现间接协同进化。

## 触发词
- 自动记忆
- 提取经验
- 记录教训
- 自动保存
- auto memory
- extract lessons

## 类型
被动技能 - 在会话后台自动运行

## 实现原理
1. 在 CLI 启动时自动初始化
2. 监控会话消息
3. 在会话结束时自动分析
4. 提取有价值的洞察
5. 更新共享记忆

## 输出格式
```
[AUTO_MEMORY] 会话开始，记录消息...
[用户输入...]
[AI 响应...]
[AUTO_MEMORY] 会话结束，提取经验...
[AUTO_MEMORY] ✅ 提取 N 条经验，已更新共享记忆
```
