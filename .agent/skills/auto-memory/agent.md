# Auto Memory

Automatically extracts lessons from individual CLI sessions and updates shared memory for indirect co-evolution.

## Domain
Cross-CLI memory and knowledge sharing.

## What It Does
Runs passively in the background during CLI sessions. Monitors messages, analyzes the conversation on session end, extracts valuable insights, and writes them to shared STIGMERGY.md memory. Enables lessons learned by one CLI to benefit others.

## When to Use
- Automatically activated on CLI startup
- No user trigger needed - passive skill
