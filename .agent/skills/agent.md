## Finding a Skill

Use `search-skill` to find relevant skills by keyword, or browse this directory and read each `SKILL.md` directly.

## Skill Format

Each skill follows the OpenSkills spec with:
- `SKILL.md` - Definition and usage guide
- `scripts/` - Executable helper scripts (optional)
- `references/` - Documentation for on-demand loading (optional)
- `assets/` - Files used in output (optional)
