How It Works

Stigmergy Collaboration Mechanism

  1. Project Context Initialization

    Automatically creates PROJECT_SPEC.json when using collaboration for the first time in project directory

  2. Collaboration Intent Recognition

    Claude recognizes collaboration intent in "let gemini help me translate"

  3. Task Creation to Context

    Creates collaboration task record in PROJECT_SPEC.json

  4. Intelligent Routing Execution

    Automatically routes to gemini to execute translation task

  5. Status Update and Synchronization

    Updates task status to shared context file for subsequent collaboration

Installation Guide

One-Click Installation
# Clone project
$ git clone https://github.com/ptreezh/stigmergy-CLI-Multi-Agents.git
$ cd smart-cli-router
$ python INTERNAL_CLI_ENHANCER.py
# Configure PATH priority
# Windows: Add %USERPROFILE%\bin to PATH beginning
# Unix/Linux: Add ~/bin to PATH beginning

After installation, all supported CLI tools will gain collaboration capabilities:

  • Claude, Gemini, Kimi, Qwen, etc.
  • Ollama, CodeBuddy, QoderCLI, etc.
  • Automatically discovers installed tools in system
