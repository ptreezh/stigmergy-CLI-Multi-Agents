# 智能体协作系统网站 - JavaScript包

为了网站正常运行，需要以下JavaScript包：

## Bootstrap Bundle
- **文件**: bootstrap.bundle.min.js
- **用途**: UI组件、导航、响应式布局
- **版本**: 5.3.0

## Font Awesome
- **文件**: fontawesome-all.min.js
- **用途**: 图标库
- **版本**: 6.0.0

## 主要脚本
- **main.js**: 网站主要功能脚本

这些包通过CDN链接在网站中引用：

```html
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/js/all.min.js"></script>
```

网站使用了现代前端技术，包括：
- Bootstrap 5 - 响应式UI框架
- Font Awesome - 图标库
- 原生JavaScript - 交互功能
