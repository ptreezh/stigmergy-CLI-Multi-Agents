安装指南

为了让智能体协作系统正常工作，您需要先安装必要的AI工具和环境。

重要提示: 如果您已经安装了iFlow，可以直接使用iFlow命令安装其他工具！

安装方式选择

方式一：命令行逐个安装

手动在命令行中依次安装下列工具：

1. 安装 Node.js

Node.js是所有AI工具的基础运行环境

# Windows下载安装包
# https://nodejs.org/dist/v24.11.1/node-v24.11.1-x64.msi
# 或使用winget
winget install OpenJS.NodeJS.LTS
2. 安装 WinGet (Windows)

对于Windows系统，WinGet用于管理系统工具

# Windows 10/11 自带WinGet，无需额外安装
# 检查WinGet版本
winget --version
3. 安装 iFlow CLI

iFlow是协作系统的重要组件

npm install -g @iflow-ai/iflow-cli@latest
4. 安装 ZCF 工具

ZCF用于一键设置Claude Code环境（API URL/KEY/Model等）

# 通过zcf一键设置Claude Code环境
npx zcf
5. 安装 Claude Code CLI

Anthropic Claude的命令行工具

npm install -g @anthropic-ai/claude-code
# 使用zcf设置环境参数
npx zcf
6. 安装 Qwen CLI

阿里通义千问的命令行工具

npm install -g @qwen-code/qwen-code
7. 安装 QoderCLI

代码生成工具

npm install -g qoder-cli
8. 安装 CodeBuddy CLI

代码助手工具

npm install -g codebuddycli
9. 安装其他AI工具（可选）
# Gemini
npm install -g @google/gemini-cli
# Kimi
npm install -g kimi-cli
# Ollama（本地模型）
npm install -g ollama-cli
方式二：通过iFlow全局安装

如果您已经安装了iFlow，可以让iFlow自动安装所有必要的工具！

智能安装: 使用iFlow的全局安装功能，它会根据需要安装其他AI工具
在iFlow中执行以下提示词：
iflow "请帮我安装智能体协作系统所需的所有工具：Node.js、WinGet、iFlow、ZCF、Claude CLI、Qwen CLI、QoderCLI、CodeBuddy CLI等"

或按步骤执行：

1. 安装环境检查工具
iflow "帮我检查系统环境，确保Node.js和npm已正确安装"
2. 让iFlow管理工具安装
iflow "请帮我安装以下工具：Claude、Qwen、Qoder、CodeBuddy、Gemini等AI CLI工具"
3. 配置工具环境
iflow "使用zcf配置Claude Code的API密钥和模型设置"
4. 安装协作系统
# 逐步安装协作系统组件
iflow "检查系统环境并验证所有AI工具的安装状态"
iflow "帮我创建 PROJECT_SPEC.json 和 PROJECT_CONSTITUTION.md 模板"
iflow "配置CLI工具的协作钩子，使其支持自然语言任务委派"
iflow "设置基于背景的间接协同（Stigmergy）机制"

安装完成验证

安装完成后，验证所有工具是否可用：

# 检查各工具版本
node --version
iflow --version
npx zcf --version # (如果安装了zcf)
claude --version # (如果安装了claude)
qwen --version # (如果安装了qwen)
codebuddy --version # (如果安装了codebuddy)

开始使用协作功能

安装完成后，您就可以在任何CLI工具中使用自然语言协作功能：

# 在任意目录中
claude "让gemini帮我翻译这份文档"
gemini "请codebuddy优化这段代码"
qwen "用claude分析这个算法的复杂度"
iflow "让qwen写代码，让codebuddy审查，让gemini写文档"
注意: 某些工具（如Claude、Gemini）可能需要API密钥才能正常工作。请参考相应工具的官方文档进行配置。
