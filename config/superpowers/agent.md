# Superpowers Configurations

Soul evolution and autonomous agent loop configurations:
- `soul-evolution.js` - Regular evolution workflow (pattern analysis, knowledge extraction, skill creation, verification)
- `soul-reflection.js` - Deep reflection after 5 evolution cycles
- `soul-co-evolve.js` - Dual-agent collaborative learning
- `soul-compete.js` - Dual-agent competitive evaluation
- `two-agent-loop.js` - Base execution mechanism for all evolution patterns
- `using-superpowers.js` - Stigmergy superpower activation

These drive the autonomous soul evolution system.
