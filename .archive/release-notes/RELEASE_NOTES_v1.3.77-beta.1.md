# Stigmergy CLI v1.3.77-beta.1 发布说明

## 发布概述

此版本包含了对 `resumesession` 技能的重要增强功能，改善了用户体验和会话历史管理。

## 主要功能更新

### 1. ResumeSession 技能增强

- **智能累积机制**：当会话内容不足时，自动追加更多会话直到达到足够的上下文信息
- **内容过滤**：自动过滤掉无意义的内容，如API限制错误、重复的技能调用、简单问候等
- **格式优化**：只显示用户输入、模型输出和时间戳信息，去除冗余格式
- **日期分组**：按日期分组显示会话，标注每组的起始和结束时间
- **无会话处理**：当当前目录没有历史会话时，直接返回"无"

### 2. 技术改进

- 改进了 `skills/resumesession/independent-resume.js` 中的会话解析逻辑
- 添加了内容过滤和去重功能
- 优化了输出格式，使信息更清晰易读

## 文件变更

- `skills/resumesession/independent-resume.js` - 增强会话解析和过滤逻辑
- `RESUMESESSION_ENHANCEMENT_SUMMARY.md` - 记录改进详情
- `CHANGELOG.md` - 更新发布日志

## 用户体验改进

1. **更简洁的输出**：只显示有价值的信息，去除噪音
2. **智能上下文获取**：自动获取足够的历史信息供AI参考
3. **更好的可读性**：按日期分组，清晰的时间范围标识
4. **快速反馈**：如果没有会话历史，立即返回"无"
