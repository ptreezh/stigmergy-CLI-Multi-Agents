## Wiki协同系统

欢迎使用Wiki协同编辑系统！

### 快速开始

1. 使用CLI命令创建新主题：
   ```bash
   stigmergy call wikiskill "创建新主题：[主题名称]"
   ```

2. 编辑现有主题：
   ```bash
   stigmergy call wikiskill "参与[主题名称]词条编辑"
   ```

3. 查看Wiki：
   ```bash
   stigmergy wiki open
   ```

### 功能特性

- 🔍 智能词条查找
- 🎭 专业角色自动设置
- 📚 知识搜索学习
- 🤔 自主思考消化
- 💬 协同编辑
- 🔄 反馈争辩反思

### 目录说明

- `topics/`: 各个主题的Wiki文件
- `config/`: 配置文件
- `assets/`: 静态资源

### 技术栈

- Wiki引擎: TiddlyWiki
- 部署方式: 本地单页文件
- CLI集成: 完全嵌入CLI生态
