## 使用方法

### 1. 直接使用生成的路由器

#### ai 路由器
```bash
# Windows
ai_router.cmd 用claude写代码

# Linux/macOS
./ai_router.cmd 用gemini分析问题
```

#### ai 路由器
```bash
# Windows
ai_router.py 用claude写代码

# Linux/macOS
./ai_router.py 用gemini分析问题
```

#### smart 路由器
```bash
# Windows
smart_router.cmd 用claude写代码

# Linux/macOS
./smart_router.cmd 用gemini分析问题
```

#### smart 路由器
```bash
# Windows

# Linux/macOS
./smart_router.py 用gemini分析问题
```

### 2. 自定义生成路由器
```bash
python universal_cli_setup.py --config config.json --cli mytool --format cmd
```

### 3. 检查工具状态
```bash
python universal_cli_setup.py --config config.json --list
```
