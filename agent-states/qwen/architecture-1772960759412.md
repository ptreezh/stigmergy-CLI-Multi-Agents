**Stigmergy** 是一个基于**痕迹刺激**（Stigmergy）概念的多 AI Agent 协作系统，通过共享记忆和痕迹标记实现跨 CLI 工具的协作与进化。

1. **痕迹驱动协作** - 通过 `STIGMERGY.md` 等共享文件实现跨 CLI 通信

### 三、识别的问题 🔴

#### 1. **版本碎片化**

版本不一致可能导致行为差异和兼容性问题。

#### 2. **代码重复**
- `hooks/auto-memory-hook.js` 与 `extensions/auto-memory/index.js` 功能高度重叠（约 80% 代码重复）
- 多处安全逻辑分散（`qwen_safe_executor.js`、`qwen_config_patcher.js`）

- `CROSS_CLI_GUIDE.md` 仅简单说明 `stigmergy distcli` 命令
