# Soul Engine

Autonomous evolution system inspired by OpenClaw. Drives continuous self-improvement of the Stigmergy system.

## Components

- `SoulEngine.js` - Core engine orchestrating the evolution cycle

## Evolution Cycle

1. **Memory** - Dual-source memory (Markdown files + SQLite-vec vector store)
2. **Reflection** - Self-critique and error analysis
3. **Skill Discovery** - Identifies new capability gaps
4. **Knowledge Production** - Multi-agent debate and consensus computation
