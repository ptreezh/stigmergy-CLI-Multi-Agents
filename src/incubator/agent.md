# Incubator - Experimental Features

Sandbox for experimental features before they are promoted to core.

## Components

- `incubator-engine.js` - Main engine integrating IM interaction, skill registry, CLI generation, progress tracking
- `im-interaction.js` - IMInteractionManager for chat platform interactions
- `skill-registry.js` - SkillRegistry for experimental skill management
- `progress-tracker.js` - IncubationProgressTracker for monitoring experiment status
- `cli-generator.js` - CLIGenerationEngine for generating new CLI tools
- `skill-orchestrator.js` - SkillOrchestrator for orchestrating skill experiments
- `builtin-cli-generator.js` - BuiltinCLIGenerator for default CLI generation
- `soul-state-manager.js` - SoulStateManager for soul state tracking

## Purpose

Rapidly prototype new features (CLIAnything integration, IM interaction, skill orchestration) without impacting core stability.
