## 🎓 技术亮点

### 1. 指数退避重连算法
```javascript
delay = min(
  currentDelay * 2,  // 指数增长
  maxDelay (60秒)     // 上限限制
)
```
**优势**: 避免频繁重试，减轻服务器压力，提高成功率

### 2. 多Bot管理架构
```
Hub
  ├── 共享凭证模式（一次扫码）
  └── 独立凭证模式（多次扫码）
```
**优势**: 灵活适配不同使用场景

### 3. 事件驱动设计
```javascript
client.on('connected', () => { ... });
client.on('disconnected', () => { ... });
hub.on('bot-added', (data) => { ... });
```
**优势**: 解耦组件，易于扩展

### 4. 消息队列管理
```javascript
await messageQueue.enqueue(async () => {
  return await client.sendMessage(...);
});
```
**优势**: 顺序处理，错误隔离，并发控制
