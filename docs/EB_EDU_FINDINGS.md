# eb-edu 研究发现

## 国家职业技能标准调研

### 电子商务师国家职业技能标准（2021版）

**等级划分**：
- 五级/初级工：基础执行层（商品信息采集、网店开设、客服、数据处理）
- 四级/中级工：独立操作层（商品管理、营销管理、订单管理、推广）
- 三级/高级工：运营策划层（营销策划、数据诊断、团队指导）
- 二级/技师：战略管理层（商业模式、全渠道规划、供应链）
- 一级/高级技师：行业专家（标准制定、培训体系）

**考核方式**：
- 理论知识考试 + 技能考核（百分制，60分合格）
- 二级及以上增设综合评审/论文答辩

### 1+X网店运营推广证书

**初级**：
- 考核模块：理论+实操（闭卷机考）
- 工作任务：商品上传维护、营销活动设置、订单管理、页面设计、客户处理
- 技能要求：网店基础运营能力

**中级**：
- 侧重：网店推广、信息流优化、流量运营、SEO、竞价推广
- 技能要求：网店规划与推广能力

**高级**：
- 侧重：网店店长、商品运营、活动运营、数据运营、店铺运营
- 技能要求：网店综合运营能力

### 教育部电子商务专业教学标准

**实训场所要求**：
- 对接真实职业场景
- 配备电商实训软件、直播设备、数据分析工具
- 环境符合教育部标准

**教学方式**：
- 情景教学、角色扮演、实操训练、项目驱动
- 工学结合，充分利用网络资源和合作企业资源

## 顶级实训导师关键洞察

### 核心原则
1. 先有真实场景，再有教学案例
2. 能力评估靠数据不靠感觉
3. 一堂好课的标准是"做得到"不是"讲得好"
4. 设计任务让学生自己做（设计合理的坑）
5. 能力认证必须有前后对比

### 关键指标（5个最看重）
1. 任务完成时间（效率提升最直观）
2. 操作错误率（反映理解深度）
3. 独立完成任务数（反映能力广度）
4. 数据决策准确率（反映分析能力）
5. 复购/回头率（反映服务质量，真实商业指标）

### 最大风险
- 变成文档系统（SKILL.md完善但无法实操）
- 数据孤岛（各技能各自采集无法汇总）
- 教师不用（备课工具太难用）
- 证书无认可（企业不认）

## 用户画像（确认）

**双角色**：
- 教师：中职/高职电商专业教师，需求是备课、授课、考核
- 学员：电商专业学生，需求是课堂实训、课后练习、技能认证

**核心价值主张**：
- 教师：AI辅助备课→一键生成实训案例→自动批改考核
- 学员：IM对话即可实训→真实Medusa后台操作→数据分析决策训练→能力认证
