## 📊 对齐度 (Alignment) 详解

### 什么是对齐度？

**对齐度**是衡量 Stigmergy Soul 系统与 OpenClaw 在各个能力维度上匹配程度的指标。

### 计算公式

```javascript
对齐度 = (Stigmergy 能力 / OpenClaw 能力) × 100%

示例计算:
技能生态对齐度 = (4 / 13,729) × 100% = 0.03%
网页自动化对齐度 = (功能完整度 × 性能匹配度) × 100%
                = (95% × 90%) × 100% = 85.5%
总体对齐度 = Σ(各维度对齐度) / 维度总数
```

### 对齐度等级标准

**🟢 高对齐度 (80%+)**
- 功能基本完整
