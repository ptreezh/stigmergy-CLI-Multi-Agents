# Stigmergy 项目宣传与 SEO 计划

## 项目概述
**Stigmergy CLI** - 多 AI 代理跨 CLI 工具协作系统

- **当前版本**: 1.10.10-beta.0
- **核心功能**: 协调多个 AI CLI 工具（Claude、Gemini、Qwen、iFlow、QoderCLI 等）协同工作
- **目标用户**: 开发者、研究人员、需要多 AI 协作的专业人士
