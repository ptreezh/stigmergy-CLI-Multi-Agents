## 🔒 安全保障机制

### 多层防护
```
用户请求skill推荐
    ↓
第1层: 静态分析（代码检查）
    ↓ (>=90分)
第2层: 动态监控（行为检查）
    ↓ (>=90分)

    ↓ (>=90分)

    ↓ (>=90分)
最终推荐
```

- 检测到异常 → 自动隔离
- 评分<90 → 拒绝推荐
- 发现安全问题 → 立即下架
- 行为异常 → 触发警报
