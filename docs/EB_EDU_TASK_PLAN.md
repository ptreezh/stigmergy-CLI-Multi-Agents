# eb-edu 实训平台开发计划（校正版）

**项目**: 跨境电商实训专业技能包（eb-edu）
**使命**: 提升电子商务网络零售课程专业实训，帮助教师开展AI时代实训教学
**对标标准**: 电子商务师国家职业技能标准（五级→三级）+ 1+X网店运营推广证书
**校正日期**: 2026-04-08
**校正原因**: AI时代实训核心是"专业能力"而非"操作效率"

---

## 🔴 根本性校正

### 错误理解（已废弃）
```
❌ 考核操作效率：商品上架32分钟→4分钟
❌ 考核操作错误率：错误25%→3%
❌ operation-tracker记录操作耗时
```

### 正确理解（新方向）
```
✅ 考核商业理解力：理解电商运营逻辑
✅ 考核数据分析力：从数据中发现问题、提出建议
✅ 考核决策判断力：基于数据做合理决策
✅ 考核问题诊断力：快速定位问题根因
✅ 考核创新思维力：提出新的运营思路
✅ 考核沟通表达力：清晰汇报运营方案
```

---

## 核心原则（来自顶级实训导师）

1. **AI帮操作，学生学专业** - 不考核操作速度，考核专业理解深度
2. **能力评估靠案例不靠操作** - 通过决策场景评估专业能力
3. **一堂好课的标准是"理解深"** - 学生能解释"为什么这么做"
4. **设计决策场景让学生思考** - 不是"怎么做"而是"为什么"
5. **能力认证必须有前后对比** - 专业能力雷达图对比

---

## 三大优先级任务（校正后）

### P0-1: 创建专业能力评估框架（3天）
**做**: 6维度评估标准 + 评分规则 + 雷达图生成
**验**: 生成1份完整的专业能力评估报告
**交付**: assessment-framework技能 + 样例报告

### P0-2: 创建前测题库（3天）
**做**: 30道题（商业理解+数据分析+决策判断）
**验**: 10人测试，能生成个人专业能力雷达图
**交付**: pre-assessment技能 + 题库

### P0-3: 创建后测+对比报告（2天）
**做**: 30道题（同维度不同场景）+ 前后对比算法
**验**: 生成10份能力提升对比报告
**交付**: post-assessment技能 + 对比报告模板

## 已废弃的内容

| 原内容 | 废弃原因 | 替代方案 |
|--------|----------|----------|
| operation-tracker操作耗时记录 | AI帮操作了，不考核速度 | decision-tracker决策路径记录 |
| CLI命令手册（P0） | 学生不需要记命令 | 降低到P2，作为AI参考 |
| 操作错误率统计 | AI帮避免错误 | 改为决策质量评估 |
| 商品上架时间对比 | 无意义（AI瞬间完成） | 定价策略理解深度对比 |
