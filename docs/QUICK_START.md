# ⚡ 5分钟快速开始

## 🎯 目标
将您的多个AI CLI工具集成为一个智能路由系统，支持自然语言调用。

## 🚀 快速步骤

### 1. 检查环境
```bash
# 检查已安装的AI工具
python universal_cli_setup.py --list
```

### 2. 生成路由器
```bash
# 为您的CLI工具生成智能路由器 (Windows)
python universal_cli_setup.py --cli mycli --format cmd

# Linux/macOS用户
python universal_cli_setup.py --cli mycli --format bash
```

### 3. 开始使用
```bash
# 智能路由 - 无需手动指定工具
smart_mycli.cmd 用claude写代码
smart_mycli.cmd 用gemini分析问题
smart_mycli.cmd 用kimi写文章
```

## ✨ 就这么简单！

- 理解自然语言指令
- 自动选择合适的AI工具
- 跨平台工作
- 支持无限扩展
