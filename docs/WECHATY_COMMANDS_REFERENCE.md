## 📋 完整命令列表

### 1. CLI 管理 🤖

#### 切换 CLI

```
切换 [cli名] cli
使用 [cli名] cli
switch to [cli] cli
use [cli] cli
```

**示例**:
```
切换 claude cli
使用 gemini cli
switch to qwen cli
use codebuddy cli
```

**可用的 CLI**:
- `claude` - 综合能力最强 (代码 0.95, 推理 0.95, 对话 0.90)
- `gemini` - 多模态专家 (多模态 0.95, 对话 0.90, 推理 0.85)
- `qwen` - 中文优化 (中文 0.95, 代码 0.85, 对话 0.90)
- `iflow` - 对话优化 (对话 0.85, 中文 0.90)
- `codebuddy` - 代码专家 (代码 0.90, 推理 0.80)
- `opencode` - 代码专家 (代码 0.92, 推理 0.85, 对话 0.80)
- `kilocode` - 代码助手 (代码 0.88, 推理 0.82, 中文 0.85)

**响应示例**:
```
✅ 已切换到 GEMINI CLI
当前 CLI: gemini
```

### 2. 任务执行 🎯

#### 执行编程任务

```
执行 [任务描述]
运行 [任务描述]
execute [task]
run [task]
```

**示例**:
```
执行 创建一个快速排序函数
运行 写一个Python脚本来解析JSON
execute 分析这个算法的时间复杂度
run 实现一个二叉树遍历
```

**智能 CLI 选择**:
系统会根据任务类型自动选择最合适的 CLI：
- 代码任务 → Claude 或 CodeBuddy
- 中文任务 → Qwen
- 多模态任务 → Gemini
- 推理任务 → Claude

**执行流程**:
1. 解析任务描述
2. 智能选择最佳 CLI
3. 执行 `stigmergy call [cli] [task]`
4. 返回结果

**响应示例**:
```
🤖 正在使用 CLAUDE 执行任务...

任务: 创建一个快速排序函数

请稍候...
✅ 任务完成

[生成的代码]
```

## 🎮 高级功能

### 智能会话管理

- **上下文记忆**: 记住最近 10 条消息
- **CLI 偏好**: 记住每个用户选择的 CLI
- **会话超时**: 30 分钟不活动自动清理
- **多用户隔离**: 每个用户独立的会话

### CLI 能力矩阵

| CLI | 代码 | 推理 | 对话 | 中文 | 多模态 | 综合分 |
|-----|------|------|------|------|--------|--------|
| Claude | 0.95 | 0.95 | 0.90 | 0.80 | 0.85 | 5.10 |
| Gemini | 0.80 | 0.85 | 0.90 | 0.85 | 0.95 | 4.96 |
| Qwen | 0.85 | 0.80 | 0.90 | 0.95 | 0.75 | 5.01 |
| iFlow | 0.75 | 0.75 | 0.85 | 0.90 | 0.70 | 4.78 |
| CodeBuddy | 0.90 | 0.80 | 0.85 | 0.80 | 0.75 | 4.73 |
| OpenCode | 0.92 | 0.85 | 0.80 | 0.75 | 0.70 | 4.87 |
| Kilocode | 0.88 | 0.82 | 0.78 | 0.85 | 0.72 | 4.80 |

### 任务特征检测

系统会自动检测任务特征并选择最佳 CLI：

1. **代码任务**: 包含"代码"、"函数"、"算法"等关键词
2. **推理任务**: 包含"分析"、"解释"、"为什么"等关键词
3. **中文任务**: 包含中文字符
4. **对话任务**: 短文本（< 50字符）
5. **多模态任务**: 包含"图片"、"文件"等

## 🔧 配置选项

### 默认 CLI

默认使用 `claude`，可以在代码中修改：

```javascript
const CONFIG = {
  defaultCLI: 'claude',  // 修改这里
  ...
};
```

### 可用 CLI 列表

```javascript
availableCLIs: ['claude', 'gemini', 'qwen', 'iflow', 'codebuddy', 'opencode', 'kilocode']
```

### CLI 能力配置

可以在代码中调整 CLI 的能力评分：

```javascript
cliCapabilities: {
  claude: { code: 0.95, reasoning: 0.95, conversation: 0.90, chinese: 0.80 },
  ...
}
```

## ⚠️ 注意事项

### 任务执行

- **超时时间**: 60 秒
- **超时处理**: 自动终止，但任务可能仍在后台运行
- **结果长度**: 超过 1500 字符会截断

### 会话管理

- **超时时间**: 30 分钟不活动
- **上下文大小**: 最多 10 条消息
- **自动清理**: 每 5 分钟清理一次过期会话

### CLI 切换

- **即时生效**: 切换后立即生效
- **持久化**: 在会话期间保持
- **智能选择**: 任务执行时会智能选择最佳 CLI
