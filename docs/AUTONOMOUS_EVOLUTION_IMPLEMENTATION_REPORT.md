**文件**: `.stigmergy/hooks/cron/evolution-schedule.js`

**核心能力**:
- ✅ 每日反思（凌晨2点）
- ✅ 每周进化（周日凌晨2点）
- ✅ 每月战略评估（每月1号凌晨2点）
- ✅ 技能发现（每6小时）
- ✅ 快速检查（每小时）

**调度机制**:
```javascript
// 每日反思
schedule('daily-reflection', '0 2 * * *', async () => {
  await runEvolutionTask('daily-reflection', {
    focus: 'mission-alignment',
    scope: 'daily'
  });
});

// 每周进化
schedule('weekly-evolution', '0 2 * * 0', async () => {
  await runEvolutionTask('weekly-evolution', {
    focus: 'capability-enhancement',
    scope: 'weekly'
  });
});
```

**状态管理**:
- 保存调度状态
- 记录执行日志
- 优雅退出机制
- 进程保持运行

- `focus`: 执行焦点（使命对齐/能力增强/最佳实践）
