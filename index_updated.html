快速开始

🚀 一步或两步完成安装

1

检查/安装 Node.js

💡 如果已安装 Node.js，可跳过此步骤 - 在终端运行 node --version 检查

📦 访问 Node.js 官网下载
📋 查看各平台详细安装指引
🪟 Windows 用户
• 下载 Windows 安装程序 (.msi)
• 选择 x64（64位）或 x86（32位）
• 右键 PowerShell → "以管理员身份运行"
🍎 macOS 用户
• 下载 macOS 安装程序 (.pkg)
• 选择 x64（Intel）或 ARM64（M1/M2/M3）
• 双击安装包按提示操作
🐧 Linux 用户
• 下载 Linux 二进制文件 (.tar.xz)
• 选择对应架构：x64、ARM64 或 ARMv7
• 解压到 /usr/local 目录
⚡ 命令行安装（服务器环境）
🍎 macOS: brew install node
🐧 Linux: curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash - && sudo apt-get install -y nodejs
🪟 Windows: winget install OpenJS.NodeJS.LTS
💡 安装后可配置国内镜像: npm config set registry https://registry.npmmirror.com
2

安装 Stigmergy

在终端/命令行中运行以下命令：

# macOS / Linux 用户
sudo npm install -g stigmergy@beta
# Windows 用户
npm install -g stigmergy@beta
💡 Windows 提示：右键 PowerShell → "以管理员身份运行"

✅ 安装完成后，运行 stigmergy --version 验证安装是否成功

Stigmergy CLI（CommandLineInteraction命令行交互） - 智能部署终端
$ sudo npm install -g stigmergy@beta
# Windows用户: npm install -g stigmergy@beta (右键 PowerShell → "以管理员身份运行")

⚡ CLI（CommandLineInteraction命令行交互）相比网页版的核心优势

🚀

极致效率

无需切换浏览器标签页，直接在终端操作。支持快捷键、命令历史、自动补全，开发者的自然工作流。

🔄

自动化集成

可编写脚本自动化调用，集成到 CI/CD 流程。网页版需要手动操作，无法批量处理。

📁

项目上下文感知

CLI（CommandLineInteraction命令行交互）直接访问本地文件系统，自动理解项目结构。网页版需要手动上传文件，且有大小限制。

🤝

多工具协同

Stigmergy 实现多个 AI 工具间的智能协作，共享上下文。网页版各工具完全独立，无法协同。

🔒

数据隐私

本地运行，敏感代码不离开本机。可配置代理、自建服务。网页版数据必须上传到云端。

⚙️

高度可定制

支持自定义 hooks、插件、技能扩展。可修改配置文件精细控制。网页版功能固定，无法扩展。

🎯 核心差异：CLI（CommandLineInteraction命令行交互）是为专业开发者和生产力场景设计，
网页版适合临时使用和轻量级任务。
