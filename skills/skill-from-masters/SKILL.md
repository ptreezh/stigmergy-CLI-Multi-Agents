## Overview

Skill from Masters provides access to expert research methodologies for advanced analysis tasks.

## Available Methodologies

### 1. Grounded Theory (扎根理论)

- Open coding (开放编码)
- Axial coding (轴心编码)
- Selective coding (选择式编码)
- Memo writing (备忘录撰写)
- Theoretical saturation (理论饱和度检验)

### 2. Actor-Network Theory (行动者网络理论)

- Actor identification (参与者识别)
- Network construction (关系网络构建)
- Translation process (转译过程追踪)
- Network dynamics (网络动态分析)

### 3. Field Analysis (布迪厄场域分析)

- Field boundary identification (场域边界识别)
- Capital distribution analysis (资本分布分析)
- Autonomy assessment (自主性评估)
- Habitus pattern analysis (习性模式分析)

### 4. Business Ecosystem Analysis

- Stakeholder mapping
- Value network analysis
- Competitive positioning

### 5. Digital Transformation

- Technology adoption assessment
- Organizational change analysis
- Digital maturity evaluation
